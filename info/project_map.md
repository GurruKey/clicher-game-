# Project Map

This document outlines the structure of the project.

## Root Directory

- `constructor/`: Python-based game configuration and constructor.
- `src/`: React-based game client.
- `info/`: Project documentation.
- `package.json`: Project dependencies and scripts.
- `vite.config.js`: Vite configuration file.
- `index.html`: Main HTML file for the React application.

## `constructor/` Directory

- `main.py`: Main entry point for the constructor application.
- `game_io/`: Modules for handling game data I/O.
- `ui/`: Flet-based UI for the constructor.
- `assets/`: Assets used by the constructor UI.

## `src/` Directory

- `main.jsx`: Main entry point for the React application.
- `App.jsx`: Root React component.
- `components/`: Reusable React components.
- `assets/`: Assets used by the game client (images, etc.).
- `config/`: Game configuration files.
- `data/`: Game data (items, locations, etc.).
- `hooks/`: Custom React hooks.
- `styles/`: CSS stylesheets.
- `utils/`: Utility functions.
